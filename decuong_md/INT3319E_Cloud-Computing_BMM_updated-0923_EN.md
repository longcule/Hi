Summary of course content</strong></p><p>This course provides comprehensive study of the concepts and capabilities of Cloud Computing across various service models including Infrastructure as a Service (IaaS), Platform as a Service (PaaS), and Software as a Service (SaaS). IaaS topics begin with details on the evolution of infrastructure migration methods from VMWare/Xen/KVM virtualization to adaptive virtualization and Cloud Computing (on-demand resource provisioning). The main Cloud infrastructure services and related vendor solutions are also covered in detail.  PaaS topics cover a wide range of platforms from Cloud service providers such as AWS, Google App Engine, Microsoft Azure, Eucalyptus, OpenStack as well as details of cross-platform services such as Google Storage, Amazon S3, Amazon Dynamo, and more providing the ability to manage and monitor Cloud resources. The SaaS and PaaS topics covered in the course will familiarize students with the use of vendor-maintained applications and processes available in the Cloud in a multi-tenant environment. The course also introduces Cloud security models and related challenges, and dives into deploying and supporting High Performance Computing and Big Data in the Cloud. Through hands-on exercises, students will learn how to configure and program IaaS services. Students will also learn how to develop software applications based on different Cloud platforms as well as how to integrate services built on heterogeneous Cloud platforms.</p><p><strong>6.
